跑得省力，才跑得持久——看看你的效率在哪里。

录一段跑步，Pose Coach 从步频、垂振、对称、步幅四个维度量化你的跑步效率，并告诉你最该优化的那一项。这测的是你跑得有多省力，不是有多快。

一个效率分，四个维度加权合成。

四个维度详解

每个维度回答三件事：测什么、好的样子、怎么改。维度顺序固定——与评分卡一致。

步频效率

权重 25%

测什么——你的跑步步频（每分钟步数）是否落在精英区间。步频太低导致每步腾空时间过长、落地冲击大；步频过高则步子过碎。

好的样子——目标区间 175–185 spm；区间内满分，偏差超过 30 spm 归零。App 内显示：步频 {v} spm · 达标 / 偏低 / 偏高。

  • 步频偏低：加入节奏专项训练（目标 175–185 步/分钟），短间歇跑可快速提频

垂振比

权重 30%

测什么——垂直振荡幅度占步幅的比值（%）。每步跳得越高、步幅越短，垂振比越高——无效弹跳多的标志，能量向上而非向前。

好的样子——精英标准 < 6%；< 8% 良好；< 10% 偏高；≥ 10% 过高（归零）。App 内显示：垂振比 {v}% · 精英 / 良好 / 偏高 / 过高。

  • 垂振过大：加强核心稳定性和髋屈肌训练，专注水平推进而非上下弹跳

儿童注意（age < 14）：ARKit 用成人骨架估算会放大儿童的足部垂振，成人阈值不适用。遇到儿童时此维度自动降级为中性分 50，并注明「未标定·儿童」。这是工具的已知限制，不是孩子跑得差。

时间对称

权重 25%

测什么——左右两腿的步态时间是否均等（各占 50%）。时间不对称暗示双侧力量或灵活性失衡。

好的样子——越接近 50% 越好；偏差 < 1.5% 为理想，< 4% 为轻微，≥ 4% 为明显不对称（偏差 10% 归零）。App 内显示：时间对称 {v}%（理想 50%）· 接近理想 / 轻微 / 明显。

  • 步态时间不对称：加入单腿专项训练（弓步蹲、单腿跳）均衡双侧力量与节律

步幅效率

权重 20%

测什么——实际步幅与「该速度下理论最优步幅」的比值。步子迈得过短（欠推进）或过长（跨步过大）都会拉低分数。

好的样子——理论最优步幅 = 配速(m/s) ÷ 3.0；实际/最优比值 1.0 满分，偏差超过 0.3 归零。App 内显示：步幅 {v} m，理论最优 {opt} m（配速 {spd} m/s），比值 {r}。

  • 步幅与配速不匹配：让配速和步频自然决定步幅，勿刻意加大跨步

参考文献：Struzik et al. 2016（步频 175–185 spm 精英参考）· Gómez-Molina et al. 2017（垂振比 < 6% 精英标准）。均为文献阈值，尚未经 Pose 跑者样本校准——见下文。

你的效率评分卡，逐元素解读

  • 评分环 + 等级徽章——四维加权综合分 0–100 及其 S/A/B/C/D 字母。
  • 等级评语——由等级生成的一句定性；B/C/D 时点名你最弱的维度。
  • 最弱维度专项练习——B/C/D 时，针对拉低分数那一维度的具体练习。
  • 四维度条——步频 / 垂振比 / 时间对称 / 步幅，各附分数与一句说明，最弱项高亮。
  • Running Power（估算）——仅在你输入体重时显示；为物理估算值，附 estimate 标注。
  • 引擎版本 + 标定状态——评分引擎版本，以及「参考值为文献预校准」的说明。

每个评分附带一句评语——以及最该练的那一项

评分给出等级评语；在 B / C / D 等级时自动附加最弱维度的专项练习——告诉你练什么，不只是「加油」。

对数字保持诚实。

效率评分参考值来自运动科学文献阈值，尚未经真实跑者样本统计校准。分数是方向性读数，反映你与文献精英参考值的偏差，不是经验证的百分制效率量表。

能做

  • 量化步频、垂振、对称、步幅四个效率维度
  • 指出最弱维度并给出专项练习
  • 估算 Running Power（需输入体重）
  • 检测数据可信度并标注异常录制

说明

  • 参考值基于文献阈值，不是 Pose 跑者样本（preliminary_ref_values）
  • 不替代运动生理测试或专业教练的周期性评估
  • Running Power 为 Minetti 2002 估算值，与 Stryd 等传感器存在偏差
  • 儿童（age < 14）的垂振比被中性化——成人阈值不适用

预校准状态

阈值来自 Struzik 2016 / Gómez-Molina 2017，尚未对 Pose 跑者样本做统计校准。

儿童（age < 14）

ARKit 成人骨架会高估儿童垂振，因此垂振比降级为中性分 50。

Running Power

由体重经 Minetti 2002 物理估算，附「estimate」标注——仅供参考，非训练负荷处方。

未采集到完整跑步时，App 会请你重录。数据可信度不足时（如步频 < 100 或 > 240 spm、垂振 > 30 cm、步幅 > 4 m）会标注「数据可信度不足」——请在光线充足、背景简洁、手机稳定水平的条件下重录。
