Pose CLI — l'analyse du mouvement pour le travail sérieux.

Une boîte à outils en ligne de commande et MCP qui transforme les données de pose 3D en rapports de mouvement reproductibles. Conçue pour les scientifiques du sport, les programmes d'éducation physique, les analystes de la performance et les équipes qui doivent livrer des résultats — pas des présentations.

$ pose analyze data/session.csv \

→ 38 frappes détectées
→ vitesse de pointe 28,4 m/s
→ indice de stabilité 0,72

$ pose chat ./data

Conçu pour celles et ceux qui étudient le mouvement

Trois groupes que nous entendons le plus.

Scientifiques du sport

Réalisez des analyses reproductibles d'une séance et d'un athlète à l'autre, exportez en Markdown / JSON pour vos articles et rapports.

Programmes d'éducation physique

Statistiques de mouvement normalisées pour des cohortes d'élèves — indicateurs de qualité, amplitude articulaire, nombre de frappes.

Analystes de la performance

Évaluation de règles propres à chaque sport (le badminton aujourd'hui, d'autres sports à venir), traitement par lots, sortie JSON adaptée à l'intégration continue.

Capacités

Des outils d'analyse enfichables dotés d'une interface cohérente.

Statistiques de mouvement

Fréquence d'images, taux de gigue, amplitude articulaire, évaluation de la qualité des données.

Détection des frappes

Multi-critères : pics de vitesse, engagement de l'épaule, cohérence de direction, filtrage de l'implication du bras.

Analyse d'énergie

Détection des pics d'énergie et profilage de la vitesse avec des fenêtres temporelles configurables.

Règles propres au sport

Conformité aux règles du badminton aujourd'hui, d'autres sports prévus. Jeux de règles personnels pris en charge.

Discussion IA (MCP)

pose chat lance Claude Code avec tous les outils d'analyse préchargés pour une exploration interactive.

Registre des lacunes

Demandes de capacités en mode participatif ; implémentations temporaires générées automatiquement pour les analyses non prises en charge.

Architecture

Une pile Python familière bâtie autour d'outils MCP enfichables.

  • API REST FastAPI pour les flux par lots et d'intégration continue
  • Serveur MCP avec outils d'analyse enfichables
  • PostgreSQL pour les métadonnées de séance, la rétroaction et le registre des lacunes
  • Docker sur Railway (préproduction + production), développement local sur localhost:8000
  • Un format JointData3D unique partagé avec Pose Coach

Format de données

Apportez vos propres données de pose — Pose CLI fonctionne avec notre CSV compact à 13 colonnes.

timestamp, joint, wx, wy, wz,
mx, my, mz,
ldx, ldy, ldz,
oreason, jitter

Le binaire .bin de Pose Coach est aussi accepté nativement. Sortie sous forme de rapports Markdown ou de JSON adapté à l'intégration continue.
